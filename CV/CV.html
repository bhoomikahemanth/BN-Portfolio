    • Compound document is a mixture of text and image, MRC uses a multi-layered imaging model which specifies the framework for document compression in order to improve the compression ratio.
    • The key to MRC Compression is segmentation of the compound document into 3 different layers: Foreground, Mask and Background.
    • Data is compressed and compressed data is then de-compressed, compressed data is unreadable while the de-compressed data is readable.
